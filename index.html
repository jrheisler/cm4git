📊 About CM4Git

CM4Git is a visual dashboard for GitHub repository KPIs. It includes a library of prebuilt charts for software project insights.

🧩 Included KPIs

  • Commits
  • Code Churn
  • Avg PR Merge Time
  • Issue Trends & Resolution Time
  • Contributor Activity
  • Lines Added vs Deleted
  • PR Rejection Rate
  • Most Changed Files
  • Release Frequency
  • ...and more

🔐 Why You Need a Token

GitHub limits anonymous API requests. To prevent errors and access more data, generate a token. No special permissions are needed — use it in each chart config.

🚀 Getting Started

  1. Click ➕ Add Box
  2. Enter repo name (like flutter/flutter)
  3. Paste your GitHub token
  4. Select a KPI, graph type, and time range
  5. Click Save — the chart will render instantly

💾 Data Storage

All data is stored in your browser’s localStorage. Nothing is uploaded. You can import/export your box layout using the buttons above.
